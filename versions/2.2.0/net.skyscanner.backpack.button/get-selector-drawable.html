getSelectorDrawable

fun getSelectorDrawable(normalColor: Int, cornerRadius: Float? = null, strokeColor: Int? = null, strokeWidth: Int? = null, pressedColor: Int, disabledColor: Int): Drawable (source)

Utility method to create a drawable with given selector states for normal, pressed and disabled. Uses either a RippleDrawable if above API 21 or StateListDrawable

Parameters

normalColor - required, used as main background color on the drawable

cornerRadius - optional, used as the radius on the corners (use 100 for square)

strokeColor - optional, color integer for the stroke

strokeWidth - optional, width in px for the stroke

pressedColor - required, color integer for the pressed state, defaults to a 20% darken factor

Return
Drawable
