monthFooterAdapter

open val monthFooterAdapter: MonthFooterAdapter? (source)

When provided, will be used to create footer views for the calendar.

If your implementation allows this property to be changed or your MonthFooterAdapter implementation can be updated in a way that requires the calendar to update, call updateContent after such update.
