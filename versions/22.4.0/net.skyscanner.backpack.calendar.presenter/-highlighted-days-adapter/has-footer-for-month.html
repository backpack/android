Overrides MonthFooterAdapter.hasFooterForMonth

Indicate that a month/year combination has a footer view.

If this method returns true for the month/year combination onCreateView and onBindView will be called to create/bind the view.

Parameters

month - The current month

year - The current year

Return
True if the month has a footer or false otherwise.
