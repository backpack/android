rawSnackbar

val rawSnackbar: Snackbar (source)

Provides access to internal Snackbar instance. This method should only be used for accessing properties and methods that BpkSnackbar does not have.

Neither of Snackbar.setAction, Snackbar.setActionTextColor, Snackbar.setText should be used with the instance.
