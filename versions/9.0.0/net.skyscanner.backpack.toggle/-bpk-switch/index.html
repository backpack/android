open class BpkSwitch : SwitchCompat (source)

BpkSwitch allow users to toggle between two states, on or off.

This class extends SwitchCompat directly and thus follows the same interface and design, with the exception of SwitchCompat.getTrackTintList and SwitchCompat.getThumbTintList that are set according to Backpack's design.
