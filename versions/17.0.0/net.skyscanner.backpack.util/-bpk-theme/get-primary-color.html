getPrimaryColor

@JvmStatic fun getPrimaryColor(context: Context): Int (source)

Utility function to fetch Backpack's primary color.

The primary color can set via theming via the bpkPrimaryColor attribute.

The attribute is expected to be either a color or a color reference. In case the attribute is not found the default R.color.bpkSkyBlue will be returned.

Parameters

context - the current ui context.

Return
a color integer
