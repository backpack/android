ThemeOverlayEnforcement(context: Context)

ThemeOverlayEnforcement is a dummy ContextWrapper used to indicate to Backpack components that the theme defined inside the bpkThemeOverlay should be used to style components.

Consider this theme:

<style name="AppTheme" parent="Theme.AppCompat.Light.NoActionBar">
  <item name="colorPrimary">@color/bpkBlue500</item>
  <item name="colorPrimaryDark">@color/bpkBlue700</item>
  <item name="colorAccent">@color/bpkGreen500</item>
  <item name="android:windowBackground">@color/bpkWhite</item>
  <item name="bpkThemeOverlay">@style/DohaTheme</item>
</style>

Now you can wrap any context during runtime with ThemeOverlayEnforcement to apply "DohaTheme" to all Backpack components.
