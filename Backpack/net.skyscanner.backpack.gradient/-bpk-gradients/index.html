Creates a GradientDrawable with the primary Backpack gradient colours by default.

For a themeable primary gradient see BpkGradients.getPrimary

Parameters

context

the current ui context.

orientation

the gradient orientation. Default value is GradientDrawable.Orientation.TL_BR

colors

the gradient colors. Default is Backpack's primary gradient.
